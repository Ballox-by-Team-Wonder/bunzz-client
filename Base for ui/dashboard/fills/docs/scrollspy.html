Scrollspy (Gumshoe)

A simple vanilla JS scrollspy script.

Gumshoe documentation

How to use

Add data-scrollspy attribute to an element for the container. Within that container any anchor link will be active when the scroller tag with the corresponding id reaches the viewport.
You can add more settings to the attribute, e.g data-scrollspy='{"offset": "30"}'. Use .scrollspy class for default styling.

By using code splitting and dynamic import techniques you don't need to worry about the size of the JavaScript file. Dashly will only import the necessary files when they are needed.
