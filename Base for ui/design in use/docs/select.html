Select (Tom Select)

A dynamic, framework agnostic, and lightweight <select> UI control.

Tom Select documentation

Basic example

Just add the data-select attribute to <select> tag.

<select class="form-select" autocomplete="off" data-select>
    <option value="Thomas Edison">Thomas Edison</option>
    <option value="John Doe">John Doe</option>
    <option value="Nikola Tesla">Nikola Tesla</option>
    <option value="Arnold Schwarzenegger">Arnold Schwarzenegger</option>
</select>

With placeholder

You can use the data-select attribute to add more settings, like placeholder data-select='{"placeholder": "Select a person..."}'. In this case you need to add an extra option tag to the select with an empty value. 

<select class="form-select" autocomplete="off" data-select='{
    "placeholder": "Select a person..."
}'>
    <option value="">Select a person...</option>
    <option value="Thomas Edison">Thomas Edison</option>
    <option value="John Doe">John Doe</option>
    <option value="Nikola Tesla">Nikola Tesla</option>
    <option value="Arnold Schwarzenegger">Arnold Schwarzenegger</option>
</select>

Select multiple

If you need to select more than one option, add the multiple attribute to the select.

<select class="form-select" multiple data-select='{
        "placeholder": "Choose members..."
    }'>
    <option value="">Choose members...</option>
    <option value="Norma Peck">Norma Peck</option>
    <option value="Perry Hart">Perry Hart</option>
    <option value="Jimmy Riley">Jimmy Riley</option>
    <option value="Martin Edwards">Martin Edwards</option>
    <option value="Katie Fowler">Katie Fowler</option>
    <option value="Zachary Ortiz">Zachary Ortiz</option>

    <option value="Lester William">Lester William</option>

</select>

Validation

It works perfectly well when you validate within a form. Read more about form validation on the Basic Forms page.

Please select a person
<form class="needs-validation" novalidate>
    <div class="mb-3">
        <select class="form-select" autocomplete="off" required data-select='{
            "placeholder": "Select a person..."
        }'>
            <option value="">Select a person...</option>
            <option value="Thomas Edison">Thomas Edison</option>
            <option value="John Doe">John Doe</option>
            <option value="Nikola Tesla">Nikola Tesla</option>
            <option value="Arnold Schwarzenegger">Arnold Schwarzenegger</option>
        </select>
        <div class="invalid-feedback">Please select your team members</div>
    </div>
    <div class="col-12">
        <button class="btn btn-primary" type="submit">Send</button>
    </div>
</form>
